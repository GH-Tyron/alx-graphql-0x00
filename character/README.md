# Character Queries

This directory contains GraphQL queries used to fetch character details
by ID using the `character(id: ID!)` field.

## Fields Retrieved
- id
- name
- status
- species
- type
- gender

## Character IDs Used
- 1
- 2
- 3
- 4

Each `.graphql` file contains a query, and the corresponding `-output.json`
file shows the expected response from the GraphQL API.
